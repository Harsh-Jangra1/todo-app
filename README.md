# 📝 To-Do List App

A modern, responsive, and user-friendly **To-Do List Application** built using **HTML5, CSS3, and JavaScript (ES6)**. It helps users manage daily tasks efficiently with features like task management, local storage, search, filters, priority levels, due dates, and theme switching.

## 🚀 Live Demo

🔗 https://your-live-demo-link.vercel.app

## ✨ Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
- 🔍 Search tasks
- 📂 Filter tasks (All, Active, Completed)
- 🎯 Priority levels (Low, Medium, High)
- 📅 Due date support
- 💾 Local Storage support
- 🌙 Dark & Light mode
- 📊 Task statistics
- 📱 Fully responsive design
- 🎨 Modern UI
- ⚡ Smooth animations
- ♿ Accessible and semantic HTML

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Google Fonts (Montserrat)

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/Harsh-Jangra1/todo-app.git
```

### Open the Project

```bash
cd todo-app
```

Open `index.html` in your browser or use **Live Server** in Visual Studio Code.
